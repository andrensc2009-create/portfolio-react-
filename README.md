# portfolio-react-
Portfólio profissional em React — páginas, componentes reutilizáveis, deploy

 [Contador de Clicks](https://andrensc2009.github.io/portfolio-react/contador-clicks/): O usuário tem três botões disponíveis: "Aumentar", "Diminuir" e "Reiniciar", e a contagem é exibida em um elemento de texto na página.
- [Contador de Palavras](https://andrensc2009.github.io/portfolio-react/contador-palavras/): Contador de palavras em tempo real em que você pode digitar no campo de entrada e ver o número de palavras e caracteres atualizados automaticamente na tela.
- [Encurtador de URL](https://andrensc2009.github.io/portfolio-react/encurtador-url/): Encurtador de URL é uma ferramenta que reduz o tamanho de uma URL, tornando-a mais curta e fácil de compartilhar.
- [Pesquisa de Filmes](https://andrensc2009.github.io/portfolio-react/pesquisa-filmes/): A pesquisa de filmes permite aos usuários buscar por filmes específicos por título, proporcionando uma experiência de usuário mais eficiente e personalizada.
